# 🖼️ Image Compressor


A lightweight and efficient image compression tool designed to optimize images without sacrificing quality. Perfect for developers, designers, and anyone who wants faster-loading websites and smaller file sizes.

---

## 🚀 Features

* 📉 Compress images while maintaining visual quality
* ⚡ Fast and efficient processing
* 🧩 Easy to integrate into any workflow
* 📂 Supports common formats (JPG, PNG, WebP)
* 🖥️ Simple and clean interface

---

## 📦 Installation

Clone the repository:

```bash
git clone [https://github.com/your-username/image-compressor.git](https://github.com/JerryMascardo/Image-Compressor.git)
cd image-compressor
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Usage

Run the compressor:

```bash
npm start
```

Or use it in your project:

```javascript
import compressImage from './compressor';

compressImage('input.jpg', 'output.jpg', {
  quality: 0.7
});
```

---

## ⚙️ Configuration

You can customize compression settings:

| Option  | Description                  | Default  |
| ------- | ---------------------------- | -------- |
| quality | Compression quality (0–1)    | 0.8      |
| format  | Output format                | original |
| resize  | Resize dimensions (optional) | null     |
